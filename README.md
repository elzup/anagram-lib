# hex-word-lib

> hex color code word dictionary

## cheat sheet

plain alphabet list

`a b c d e f`

similar alphabet number list (order by useful)

* `0 -> o`
* `1 -> i, l`
* `9 -> q, g`
* `2 -> z`
* `5 -> s`


=> `a, i, e, o` + `b, c, d, f, l, q, g, z, s`
